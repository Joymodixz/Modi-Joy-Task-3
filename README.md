Task 3 - Basic Vulnerability Scan on Local System
Objective
The objective of this task was to perform a basic vulnerability assessment on a local machine using a vulnerability scanning tool and identify potential security risks along with their mitigation strategies.

Tool Used
Nessus Essentials
Target System
Localhost (127.0.0.1)
Operating System: Windows
Steps Performed
Installed and configured Nessus Essentials.
Set the scan target as localhost (127.0.0.1).
Selected and initiated a Basic Network Scan.
Allowed the scan to run until completion.
Analyzed the detected vulnerabilities and their severity levels.
Identified the most critical vulnerabilities.
Researched possible fixes and mitigation techniques.
Captured screenshots of scan results for documentation.
Vulnerability Summary
Severity	Description
High	Missing security updates
Medium	SSH service enabled
Medium	Weak SSL/TLS configuration
Low	Open network ports
Informational	System and service information
Remediation Recommendations
Install all pending security updates immediately.
Disable unnecessary services such as unused SSH access.
Configure firewall rules to restrict unauthorized access.
Use strong authentication methods and secure credentials.
Update and maintain all system applications regularly.
Perform periodic vulnerability scans to monitor system security.
Files Included
Vulnerability_Report.txt
Scan Screenshots
README.md
Outcome
The vulnerability scan successfully identified multiple security issues within the local system. The most critical risk was outdated system updates, which could expose the system to known exploits. By applying the recommended fixes, the overall security posture of the system can be significantly improved.
